POST /api/users/

Description: Creates a new user.

Parameters: username, email, password

Request Body:

{
    "username": "new_user",
    "email": "newuser@example.com",
    "password": "secure_password"
}

Response:

{
    "id": 3,
    "username": "new_user",
    "email": "newuser@example.com"
}
